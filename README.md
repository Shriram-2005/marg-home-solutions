## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/ui/` - Reusable Shadcn UI components
- `components/shared/` - Custom application components
- `lib/` - Utility functions
- `public/` - Static assets (images, logos)

## Adding Images

Place your images in the following directories:
- `public/brands/` - Brand logos (Adani, Racold, Grundfos)
- `public/products/` - Product images
- `public/hero-bg.jpg` - Hero section background

## Customization

- Brand colors: Edit `tailwind.config.ts`
- Global styles: Edit `app/globals.css`
- Metadata: Update `app/layout.tsx`

## Build for Production

```bash
npm run build
npm start
```
